#DevOps First Step 
Учебный и мнемотехнический проект для изучения и запоминая практик, технологий. 

##Структура проекта

devops-1step/
- .gitignore # Игнорируемые файлы Git
- requirements.txt # Зависимости Python
- .env # Переменные окружения (не коммитится!)
- app/ # Бэкенд на Flask
  1. init.py
  2. app.py # основной файл Flask
  3.  db.py # работа с БД
- rontend/ # Фронтенд (статический HTML/JS)
- index.html # простой интерфейс


## Технологии, применяемые в проекте

- Backend: Python + Flask
- Database: PostgreSQL
- Frontend: HTML + Vanilla JS
- OS: Ubuntu (WSL2)
- VCS: Git + GitHub
